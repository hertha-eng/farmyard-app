# FarmYard Android QA Test Matrix

## Account And Auth

- Register with email and password
- Sign in with email and password
- Sign in with Google
- Sign out and sign back in
- Confirm OAuth callback returns to the app

## Marketplace

- Browse listings
- Search listings
- Open listing detail
- Save a listing
- Post a listing with an image

## Messaging And Calls

- Start a new conversation
- Exchange messages between two accounts
- Request notification permission where supported
- Start and receive an in-app audio call
- Deny microphone permission and confirm graceful handling

## Compliance

- Open privacy and support pages
- Confirm legal copy is reachable
- Confirm phone-number privacy settings behave as expected
