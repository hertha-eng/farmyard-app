# FarmYard Play Store Metadata Draft

## App Name

FarmYard

## Short Description

Buy, sell, and message across produce, farm inputs, and agricultural services.

## Full Description

FarmYard is an agriculture-focused marketplace built for farmers, buyers, suppliers, and service providers. Users can create listings, manage profiles, message safely, and coordinate purchases for produce, farm inputs, and agricultural services from one account across web and mobile.

## Support

- Support email: support@farmyard.app
- Privacy policy URL: https://farmyard.app/privacy.html
- Website: https://farmyard.app/support.html

## Permissions Rationale

- Microphone: used for in-app voice calling between marketplace users
- Camera and photos: used when sellers add listing images
- Network access: used for sign-in, listings, chat, and real-time updates
