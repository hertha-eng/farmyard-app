# FarmYard iPhone App Review Notes

## Reviewer Summary

FarmYard is an agriculture marketplace for listings, messaging, and account management across web and mobile. The app helps users discover produce, farm inputs, and agricultural services. Payments are not processed inside the app.

## Test Flow

1. Launch the app.
2. Register a new account or sign in with an existing account.
3. Browse marketplace listings from the home screen.
4. Open a listing and start a message thread.
5. Post a new listing with an image.
6. Open the legal section for privacy and marketplace policy text.

## Permission Explanations

- Microphone: used only if the user initiates an in-app voice call.
- Photos or media: used only when the user selects listing or profile images.

## Important Notes

- FarmYard is not a payment processor.
- Content is intended for agriculture-related trade only.
- If App Review needs a demo account, include it in App Store Connect review notes.
